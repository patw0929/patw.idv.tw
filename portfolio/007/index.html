7-ELEVEn 涼夏季哆啦A夢轉彎風扇

http://www.7-11.com.tw/event1/2011doraemon/

7-ELEVEn 推出集點商品─哆啦A夢轉彎風扇，活動網站中設計了神奇占卜遊戲，讓網友可在網上玩小遊戲後蒐集轉彎風扇，並留下資料抽獎。

由於一開始在機制上設計不良，過度的 Facebook 發布貼文與 Tag 造成應用程式一再被 Facebook 系統砍掉，甚至連開發者帳號都被停用；後來在仔細研究 Facebook 應用程式開發規範 後（也因此寫了 Blog 筆記），避免這些開發上的誤區，終讓活動運行正常。

另外，還有「涼夏合照趣」與「合格應援團」兩個子活動。「涼夏合照趣」讓網友登入 Facebook 後可將自己的照片與哆啦A夢的角色一起合照並發布至 FB。而「合格應援團」則利用 Facebook JavaScript SDK 撈出使用者互動最高的幾名好友，讓使用者能留下祝福給這些朋友。
