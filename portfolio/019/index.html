活動網站架構一鍵產生器（ASP.NET）

  • 類別自創工具
  • 時間2012/12
  • 擔任程式設計
  • 參與技術WinForm (C#) 程式設計

某個晚上在家，忽然覺得活動案的架構都大同小異（API、後台、抽獎），每次寫實在很浪費我的時間，又覺得原來寫的程式碼演變得很髒，所以就心血來潮進行「活動架構一鍵產生器」這個祕密武器的開發 :P

主要使用 Visual Studio 2010 C# 進行 WinForm 的開發，參考對岸一些程式碼自動化工具的架構，並搭配 Fluent Validation 驗證類，以及 .NET 的輕量化 ORM ─ PetaPoco。

使用方法是先連接 Database（SQL Server），設定各欄位的驗證屬性（例如長度、限制中文、檢查 E-mail 格式等）、 後台設定（分頁、欄位是否顯示、是否輸出至 Excel）、抽獎設定（機率設定等），最後將程式碼輸出到指定的資料夾中（包含 API、DataModel、Web.config、後台列表、抽獎等）， 開發者再針對個別案件的情況做程式碼調整即可。可大幅減低重作這些雜事的時間 :PP
