Burt's Bees 夏日樂園

  • AgencyiPrefer
  • ClientBurt's Bees
  • 時間2010
  • 擔任程式設計、Flash 程式設計
  • 參與技術PHP, Facebook PHP SDK, Flash ActionScript 3
活動已下線

Burt's Bees 於 Facebook 上的網路活動，營造了一個夏日樂園，園內的各項遊樂設施皆有不同的小遊戲與贈品活動。

最後加入粉絲團人數達 1 萬多人，已超出客戶原先設定 5 千人的目標。

此案中我負責的部分是將設計師提供的 FLASH 檔案加上 ActionScript 3 程式，再以 Facebook PHP SDK 開發成 Facebook 應用程式。
