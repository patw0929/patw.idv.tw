御茶園的發票登錄活動，以「暢遊世界五大洲」作為活動主題，除了讓消費者登錄發票外，還可在世界地圖上選擇希望旅遊的城市。

此案負責的部分是 ASP.NET 與 Flash Action Script 3 的程式開發；此外還試作了此活動的發票登錄 App (for Android)，使用 Titamium Mobile 進行開發。
